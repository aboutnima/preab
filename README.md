# Preab

> Smart Presence & Absence Management

Preab is an open-source presence and absence management platform designed for companies that need a simple, modern, and reliable way to track employee attendance.

The project started from a real internal need — managing employee check-ins, check-outs, shifts, and leave requests without relying on spreadsheets or manual tracking.

Preab focuses on simplicity, clarity, and usability while keeping the architecture flexible for future expansion.

---

## What is Preab?

Preab stands for:

- **Pre** → Presence
- **Ab** → Absence

It is a web-based attendance tracking system where employees can:

- Check in
- Check out
- Track work shifts
- Request leave
- View attendance history

Managers and companies can:

- Monitor active employees
- Analyze attendance data
- Generate reports
- Track workforce activity

---

## Why Preab?

Many companies still manage attendance manually or with outdated systems.

Preab aims to provide:

- A clean and modern interface
- Easy employee attendance tracking
- Real-time workforce visibility
- Accurate reporting and analytics
- A flexible foundation for future HR-related features
